# Advanced Hardcore Engineer - 核心架构与教案重构总纲

## 🎯 痛点与重构愿景 (Vision)
之前的课程存在“理论空洞、参数缺失、前后脱节”的致命弱点。为了达到真正的“专家级”标准，我们必须抛弃孤立的知识点教学。
**重构核心策略**：引入一个贯穿 48 节课的总项目 —— **「Cyber-Sorter：视觉引导式四轴分拣机械臂与AGV移动底盘」**。
每一节课都是为了完成这个机器人的某一个具体零件、某一段代码或某一项测试。参数将绝对精准，所有内容必须能真正在现实中组装运行。

---

## ⚙️ 第一阶段：机械结构与数字孪生 (Lesson 1-24)
*目标：在 SolidWorks 中完成总项目的自底向上建模与数字孪生验证，输出可供 3D 打印的 STL 文件。*

### 模块一：基础制图与底盘构建 (L1-L8)
*本模块将完成 AGV 移动底盘的基础零件设计，掌握制图基石。*
- **L1-L2 (制图基准)**：设定 SolidWorks 模板，建立坐标系。绘制 NEMA 17 步进电机标准安装座的三视图。
- **L3-L4 (草图与约束)**：绘制 AGV 底盘的主受力框架草图，精确标注尺寸（公差预留 0.2mm）。
- **L5-L6 (基础特征)**：拉伸底盘主板，旋转切除法兰孔。生成标准工程图，标注形位公差。
- **L7-L8 (剖视与测绘)**：测绘并绘制全向麦克纳姆轮的轮毂结构，使用剖视图展示内部轴承孔位（内径 5.05mm 过盈配合）。

### 模块二：高级曲面与机械臂构建 (L9-L16)
*本模块将完成四轴机械臂的连杆、齿轮箱与末端执行器（夹爪）建模。*
- **L9-L11 (复杂特征)**：使用扫描与放样设计机械臂的仿生镂空连杆（减轻重量），使用曲面过渡消除应力集中点。
- **L12 (质量属性)**：指定材质为 PETG / PLA，计算当前连杆的重心（Center of Mass）与转动惯量，为后续 PID 调参做准备。
- **L13-L15 (传动系统)**：参数化设计机械臂底座的回转减速齿轮组（模数 m=1, 齿数比 1:4），并构建螺旋传动杆。
- **L16 (3D打印制造)**：导出所有零件。在切片软件中设定：外墙 3 层，填充 25% Gyroid，机械臂悬垂角 > 50° 添加树状支撑。

### 模块三：装配验证与 CAE 仿真 (L17-L24)
*本模块进行云端装配，验证运动空间干涉，并进行受力分析。*
- **L17-L19 (系统装配)**：将底盘与机械臂部件引入装配体，应用同轴心、重合与齿轮高级配合。
- **L20-L22 (干涉与运动学)**：运行碰撞检测，确保机械臂在极限夹角（-90° 至 +90°）下不会与底盘干涉。加入马达生成运动学仿真动画。
- **L23 (有限元分析 FEA)**：在机械臂夹爪施加 5N 的垂直负载，网格化并运行静态应力分析，查看安全系数（FOS > 2.0 视为合格）。
- **L24 (里程碑)**：完成整机数字孪生验证，下发所有物理零件的 3D 打印任务。

## ⚡️ 第二阶段：硬件控制与具身智能 (Lesson 25-48)
*目标：使用 ESP32 赋予机器人神经系统，打通底层运动控制、传感器融合与机器视觉。*

### 模块四：底层硬件与执行器 (L25-L31)
*本模块打通 ESP32 与各类电机的底层驱动协议。*
- **L25-L26 (MCU与DC驱动)**：配置 PlatformIO/Arduino IDE。讲解 H 桥电路，使用 TB6612FNG 驱动 AGV 底盘的直流减速电机，输出 10kHz PWM 调速。
- **L27 (步进控制)**：连接 A4988 驱动器与 NEMA17，编写精准的脉冲控制函数（`step(int steps, int dir)`），实现底座的毫米级旋转。
- **L28 (伺服舵机)**：使用硬件定时器生成 50Hz PWM，精确控制 MG996R 与 SG90 舵机，实现机械臂关节与夹爪的闭合。
- **L29-L31 (总线与布线)**：通过 I2C 挂载 OLED 屏幕显示系统状态，规范 12V 动力电与 5V/3.3V 逻辑电的隔离布线，防止 MCU 重启。

### 模块五：控制算法与运动学解算 (L32-L40)
*本模块引入核心算法，让机器人运动更加平滑、精准。*
- **L32-L33 (传感器融合)**：解析红外循迹模块与超声波测距数据，结合外部中断触发避障逻辑。
- **L34-L35 (PID 控制)**：引入编码器反馈，手写增量式 PID 算法，实现 AGV 底盘的直线匀速行驶与平滑转弯。
- **L36-L37 (运动学解析)**：硬核数学课。构建 D-H 参数表。使用三角函数推导正向运动学（FK）与逆向运动学（IK），实现输入 (X, Y, Z) 坐标自动计算关节角度。
- **L38-L40 (通信系统)**：建立帧头帧尾指令集协议（如 `<G0 X100 Y50 Z20>`），通过蓝牙或 UART 实现上位机到 ESP32 的指令解析下发。

### 模块六：机器视觉与系统集群 (L41-L48)
*本模块引入 Python 与 OpenCV 上位机，完成最终的全自动分拣任务。*
- **L41 (系统联合)**：总装所有打印件与电子硬件，进行联调，标定机械臂的原点偏移量（Offset）。
- **L42-L43 (视觉识别)**：在 PC 端使用 Python + OpenCV。调用摄像头，转换 HSV 空间，提取色块（红/蓝木块）的轮廓与中心点图像坐标。
- **L44 (空间映射)**：将 OpenCV 的图像像素坐标映射到机械臂的物理空间笛卡尔坐标系。
- **L45-L47 (最终决战)**：打通整个流！视觉识别目标 -> 计算三维坐标 -> 上位机发送指令集 -> ESP32 接收并进行 IK 逆解 -> 规划平滑轨迹 -> 舵机与步进电机联动执行抓取并放置到指定 AGV 车厢。
- **L48 (结业路演)**：CSWA 认证标准考核，代码重构优化，项目复盘与未来技术延展（如接入 ROS）。
