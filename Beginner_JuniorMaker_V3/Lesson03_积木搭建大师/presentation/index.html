Lesson 03

空间直觉与视觉破解

告别平面的画纸，进入真正的三维世界！掌握神秘的红绿蓝方向盘，像搭乐高一样建造你的 3D 城堡。

第一重魔法：红绿蓝三维方向盘 🕹️

在数字世界里，没有手去抓方块，我们只能驾驶三架不同颜色的直升机来搬运积木。

第二重魔法：破解 2D 屏幕的视觉欺骗 👁️

从正面看，这颗球是不是稳稳地停在柱子正上方？按住下方的按钮旋转视角看看真相吧！

建筑师铁律：永远不要只从一个角度看模型，要不停地绕着它转圈看！

🚨 工程师现场灾难演习 (避坑指南)

🛸

灾难 1：悬空打印，塑料化泥

现场：切片软件底部出现橙色警告，打印出来的屋顶直接掉在地上变成一滩烂泥。

解药：对抗地心引力！每一个在半空中的零件，下方都必须有一根柱子撑着它。屋顶必须和柱子产生穿插重叠，哪怕只穿插 0.1 毫米！

🌌

灾难 2：迷失太空，模型消失

现场：用鼠标狂甩或输入了错误的放大数值，导致模型消失在屏幕的浩瀚视野外。

解药：不要手动乱转去找！在左侧的对象列表里选中你的模型，然后按下键盘上的 0 键（居中快捷键），模型就会瞬间回到屏幕正中央！
