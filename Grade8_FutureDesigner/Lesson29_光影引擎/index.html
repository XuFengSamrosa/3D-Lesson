光之工程学

两种发光路径 路径A｜镂空透光：蜂窝孔阵列让 LED 光线形成鲜明光纹，犀利如激光。路径B｜薄壁渗光：0.8mm 半透明 PETG 让整面墙体均匀散光——建筑变成发光体！
KEYWORDS / 极客密语 Lattice 晶格镂空, Translucent PETG 半透明渗光, LED 走线槽预留

参数化指令

镂空战法 六边形透孔草图(Ø8mm) + Circular Array × 环形6孔 + Linear Array × 垂直5排 = 30孔蜂窝装甲，关闭塔身填充！
渗光战法 Shell 0.8mm 薄壁 + 白色/透明 PETG + 填充率 0% 内部全空 + 底部切 4×3mm LED 走线槽。整塔均匀发光！
WARNING 镂空密度红线：孔与孔之间的实体材料最薄不得低于 3mm！低于此值每层打印时孔间肋板会因振动整面碎裂。

切片覆盖

> SETTINGS 路径A: 普通 PLA（任意色），层高 0.15mm，确保镂空孔内壁光洁无毛刺。
> SETTINGS 路径B: 白/透明 PETG，层高 0.12mm，外壁仅 2 圈（保持薄透），填充率 0%（全空腔），LED 背光即可均匀透出。

极客核心试炼

TARGET LOCK 在你的地块建造能够内置光源的建筑（路径 A 或 B 任选），底部必须有 LED 走线槽预留！ 状态检定：关灯后手机手电筒从底部照入，是否从外壳均匀透光（路径B）或从孔洞射出（路径A）？
