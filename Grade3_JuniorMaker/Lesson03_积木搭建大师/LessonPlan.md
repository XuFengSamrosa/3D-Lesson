# 3 年级 | 第 3 课：积木搭建大师 (教师手册版)

## 0. 课前准备
- **硬件**：学生电脑（已安装 Bambu Studio）、投影仪。
- **操作软件**：Bambu Studio（用于搭建基础形状组合）。
- **材料**：提前3D打印好的基础几何形状教具（包含多个立方体、圆柱体、圆锥体、球体，用于充当实物积木展示）。
- **教学案例**：准备 1 个已打印好的"迷你城堡"实物，由基础形状拼搭而成。

---

## 1. 挑战书：积木大门 (5 分钟)

### 激趣导入：
1. **实物展示**：拿出准备好的3D打印几何教具（立方体、圆柱、圆锥、球体等），在讲台上快速拼搭出一个简单的"城堡大门"。
2. **设问激趣**："同学们，老师刚才用打印出来的几何积木搭了一个小门。那如果在电脑里，我们把它们变成真正的『数字积木』，能不能搭出更酷的城堡？"
3. **挑战宣布**：今天的任务——用 Bambu Studio 里的"数字积木"搭建一座迷你城堡！
4. **展示成品**：展示已打印好的城堡模型，让孩子们产生期待。

---

## 2. 拼搭城堡：数字积木总动员 (25 分钟)

### 🔴 这是本节课的核心环节：多形状组合、移动与缩放 🔴

### 第一步：召唤积木
1. **打开 Bambu Studio**：新建空白项目。
2. **添加基础形状**：
   - 右键 -> 添加基础形状 -> **立方体**（这是我们的城墙）。
   - 再添加 **圆柱体**（这是城堡的柱子）。
   - 再添加 **圆锥体**（这是塔尖的帽子）。
   - 再添加 **球体**（这是塔楼最顶端的魔法明珠）。
   - **操作比喻**："每个形状就像一块数字乐高，我们要用它们拼出城堡！"

### 第二步：学新魔法——移动、缩放、旋转
1. **移动工具（Move）**：
   - 点击上方工具栏的"移动"按钮。
   - 拖动彩色箭头（红=X轴左右、绿=Y轴前后、蓝=Z轴上下）。
   - **练习**："把圆柱体搬到立方体的旁边，就像把柱子立在城墙边。"
2. **缩放工具（Scale）**：
   - 点击"缩放"按钮。取消勾选右侧面板的“均匀缩放”（重要环节），以便单独调整各边。
   - **城墙（原立方体）**：设定 X=60mm，Y=10mm，Z=20mm（变成一堵扁长矮墙）。
   - **柱子（原圆柱体）**：设定 X=15mm，Y=15mm，Z=40mm（变成高瘦圆柱）。
   - **塔尖（原圆锥体）**：设定 X=20mm，Y=20mm，Z=25mm（比柱子略宽，像个大帽子）。
   - **操作比喻**："缩放就像是用魔法拉面条，把方块拉成墙，圆柱拉成柱。"
3. **旋转工具（Rotate）**：
   - 点击"旋转"按钮。演示让圆柱沿着X轴或Y轴旋转 90度（让柱子"躺下"），再输入 0度或按快捷键 Ctrl+Z 撤销让它重新站起来（让学生理解90度直角旋转的概念及运用）。

### 第三步：复制与叠高组合
1. **复制材料**：目前我们只有1个柱子和1个塔尖。选中调好尺寸的柱子，使用 `Ctrl+C` 复制，`Ctrl+V` 粘贴，得到两根柱子。同样的操作，复制出一个新的塔尖，凑齐一对。
2. **位置定位**：把两根圆柱体分别移到城墙的两端，作为门柱。
3. **加帽子（叠高）**：
   - 把两个圆锥体分别移到圆柱体的正上方，作为塔尖。
   - **精准定位技巧**：因为我们刚才把柱子高度（Z尺寸）设为了40mm，所以选中塔尖，直接把右侧的**Z轴位置（高度）修改为40mm**，它就能瞬间并精准地"坐"在圆柱顶上！
   - **视觉检查**：用蓝色箭头微调，并提醒大家："一定要各个角度转动视角看一看，要让锥体刚好落在柱子顶上，不要有缝隙，也不能飘在空中哦！"
4. **组合更多**：选中一组搭好的柱子+塔尖（框选或按住Shift多选），使用 Ctrl+C / Ctrl+V 快速复制，自由摆放到城堡的其他位置。

### 🧱 概念引入：组装 vs 悬空
- **组装**：所有形状之间紧紧贴合，底部稳稳地坐在打印板上。
- **悬空**：如果一个形状飘在半空中，3D 打印机就像在空气里画画——打印出来会塌！
- **操作比喻**："大家想想，能不能在空气上面画画？不行对不对！所以我们的积木一定要脚踏实地，每一块都有东西支撑。"

## 3. 细节装点与作品展示 (15 分钟)

### 装饰城堡 (10 分钟)：
1. **挖出大门（复习负零件）**：在第二课做名牌时，我们已经学过了如何利用"负零件"挖洞！
   - 添加一个立方体作为“门洞模板”，调整为建议尺寸：**X=10mm，Y=15mm，Z=12mm**（Y轴刻意比城墙的10mm长一点，确保能完全穿透）。
   - 把门洞模板移到城墙正面正下方中央。
   - 右键将其类型修改为**“负零件”**，然后和城墙一起框选，进行**组合（布尔运算）**，就能快速挖出一个真正的城门洞！
2. **加城垛**：在城墙顶部加一些防御用的小垛口。
   - 添加一个小立方体，调整为建议尺寸：**X=4mm，Y=10mm，Z=4mm**。
   - 把它的高度（Z轴位置）设为 **20mm**，让它精准地坐在城墙上。然后复制多个小方块，等间距排列在城墙顶部，做出锯齿状的效果。
3. **摆放魔法球**：拿出我们召唤的**球体**，调整尺寸为 **X=15mm, Y=15mm, Z=15mm**，同样利用调整Z轴高度的技巧，把它稳稳地安放在城堡最中间或塔尖最高处作为魔法明珠！
4. **自由创作**：鼓励孩子在自己掌握的技巧上自由发挥——给塔楼加窗户（用小块负零件挖洞）、在旁边新建高低的围墙等。

### 作品大合影 (5 分钟)：
1. **调整视角**：教孩子们用鼠标滚轮缩放、按住鼠标中键旋转视角。
2. **截图保存**：从一个酷炫的角度截图保存自己的城堡。
3. **大屏展示**：选几位同学的作品投影到大屏上，全班一起"参观"数字城堡。

## 教师提示 (Instructor Tips)：
- **移动工具是本课的重中之重**。三年级学生对三维坐标系概念模糊，老师可以用"左右、前后、上下"来替代"X、Y、Z"的说法。
- **对齐示范要慢**。叠放圆锥到圆柱顶部时，老师要放慢操作速度，反复对比"差一点没对齐"和"对齐了"的区别。
- **悬空问题**是 3D 打印的核心认知之一，这里只需初步埋下概念种子，不用深入讲解支撑结构。
- **课后任务（可选）**：课后教师可以选择一两个完成度高的作品进行打印，在下节课上发放，作为激励。
