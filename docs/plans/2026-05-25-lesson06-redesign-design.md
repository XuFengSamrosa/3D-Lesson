# 2026-05-25 第6课个性化定制重构设计说明 (V1 & V3)

本文档规划了 V1.0 第6课《桌面小管家收纳盒》与 V3.0 第6课《城堡守卫哨所》的全新定制化展示与交互方案。方案抛弃了死板的五步法步骤控制，根据每节课的具体知识点重新设计界面风格与互动逻辑。

## 1. V1.0 第6课：桌面小管家收纳盒

### 1.1 设计定位
*   **主题/风格**：深色调“CAD 极客工程蓝图风”。界面背景为带蓝色坐标细线的工程图纸网格。
*   **交互逻辑**：取消 0-4 步骤按钮，换为自由滑动的“壁厚”与“底厚”参数调节面板。
*   **3D 展示**：展示收纳盒主体及内部掏空负零件。

### 1.2 核心交互元素
1.  **壁厚调节滑条 (Wall Thickness Slider)**：
    *   范围：`0.4mm` 到 `8.0mm`，步长 `0.2mm`，默认值 `2.0mm`。
    *   效果：拉动时，中央收纳盒的壁厚实时加宽/缩窄。
2.  **底厚调节滑条 (Floor Thickness Slider)**：
    *   范围：`0.0mm` 到 `10.0mm`，步长 `0.5mm`，默认值 `3.0mm`。
    *   效果：拉动时，内部的负零件底面在 Z 轴上做对应的移动。
3.  **X光透视开关 (X-Ray Toggle)**：
    *   效果：开启时，外壳半透明，显现内部红色的负零件形状。
4.  **动态状态警告回显**：
    *   底厚 = 0.0mm 时：高亮红色警告“🚨 灾难错误：底板厚度为 0mm！收纳盒漏底变通心管，笔插进去会直接掉出来！”。
    *   底厚 > 0.0mm 且 < 1.2mm 时：警告“⚠️ 警告：底部过薄，打印时容易损坏漏水！”。
    *   壁厚 < 1.2mm 时：警告“⚠️ 警告：壁厚过薄，打印出来的盒子一捏就碎！”.
    *   壁厚在 1.2mm - 3.0mm，底厚在 1.5mm - 5.0mm 时：显示“✅ 黄金参数，结构强度与材料消耗完美平衡，合格！”。

## 2. V3.0 第6课：城堡守卫哨所

### 2.1 设计定位
*   **主题/风格**：暗绿色军用全息激光战略沙盘风。背景是带暗绿色荧光线框的战略地图和线框网格。
*   **交互逻辑**：聚焦哨所大门切割高度与宽度的变化对城堡塔基座造成的物理卡接承重影响。
*   **3D 展示**：哨所主体带有拱门切削插口。

### 2.2 核心交互元素
1.  **大门切削高度滑条 (Gate Cut Height)**：
    *   范围：`0mm` 到 `60mm`，默认 `35mm`。
    *   效果：控制哨所下方拱门凹槽切除的深度。
2.  **门宽调节滑条 (Gate Width)**：
    *   范围：`10mm` 到 `55mm`，默认 `30mm`。
    *   效果：控制拱门凹槽的宽度。
3.  **基座断裂物理动画**：
    *   当大门高度切除过深（导致哨所顶部或基底承重厚度为 0mm，即切穿底板或大门宽度等于外盒宽度）时，哨所模型的基底板分裂为左右两半并向下倾斜，同时出现**基座断裂闪烁警报**：“🚨 战略警报：基板已被门洞切穿断裂！城防城墙拼装时将无法承重，高塔倒塌！”
    *   合格状态时显示：“✅ 哨所卡接凹槽安全锁定，城墙承重契合度 100%！”
4.  **分层切片投影扫描仪 (Slicing Scanner)**：
    *   提供一个高度扫描滑条（从 0mm 到 80mm）。
    *   滑动时，扫描光波在 3D 模型上垂直移动，右侧面板实时绘制出在该高度下的 **2D 打印切片截面投影图**（0mm 为实心方，30mm 为带拱门 U 型槽，70mm 为实心圆/方顶面），生动演绎 3D 打印分层逻辑。
