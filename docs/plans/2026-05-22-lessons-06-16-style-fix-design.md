# 2026-05-22 Lessons 06-16 Style Fix Design

本文档详细描述了为 `Beginner_JuniorMaker` (v1版本) 课件演示中第6课至第16课（Lesson 06 - 16）补全 3D 效果与样式的技术设计。

## 目标与背景
用户反馈第6课之后的演示课件存在样式缺失、3D 动画失效的问题。经过排查，发现 Lesson 06-16 的 `presentation/` 目录中缺少 `custom.css` 与 `custom.js`，且 `index.html` 尚未引入它们。
本设计采用**方案 A**，通过纯 CSS 3D 变换 (`transform-style: preserve-3d`) 以及 JS 状态切换类，为这 11 节课提供生动的 3D 步骤演示与现代科技感的外观。

## 详细技术实现

### 1. 通用技术栈与规范
* **CSS 3D 变换**：利用父容器 `.scene` 的 `perspective: 1200px`。为 `.part` 开启 `transform-style: preserve-3d`，并通过绝对定位在空间中组装 `.face`。
* **动效**：使用 CSS transition 属性 (`transition: all 1s cubic-bezier(0.25, 1, 0.5, 1)`) 实现平滑移动、旋转和缩放。
* **状态控制**：编写统一的 `custom.js`，为步骤切换按钮绑定事件，切换 `.scene` 或父级容器的 `.step-0`, `.step-1` 等类名，通过 CSS 状态选择器触发对应的动画。
* **引入方式**：
  在 `index.html` 中引入：
  ```html
  <link rel="stylesheet" href="custom.css">
  ...
  <script src="custom.js"></script>
  ```

### 2. 各课 3D 演示细节

#### Lesson 06: 桌面小管家
* **零件与面**：
  * `.main-body`：60x60x80px 3D 灰色磨砂实体盒子。包含 `front`, `left`, `top`（及其余需要的面）。
  * `.negative-core`：50x50x90px 3D 红色半透明全息实体。包含 `front`, `left`, `top`。
* **步骤过渡**：
  * `0`：初始化，仅显示网格。
  * `1`：`.main-body` 渐入并显示实心 3D 样式。
  * `2`：`.negative-core` 降落重叠在主体中心。
  * `3`：`.negative-core` 向上偏移 10px，保住底面。
  * `4`：`.negative-core` 隐藏，`.main-body` 渲染为“掏空”虚线盒样式。

#### Lesson 07: 迷你景观配件
* **零件与面**：
  * `.tub`：直径 60px、高 15px 的 3D 浅灰色圆筒底座（用 border-radius 拼合或扁平圆片表示）。
  * `.div1`, `.div2`：60x60x4px 的 3D 薄隔板。
  * `.tree`：绿色发光、科技半透明的 3D 支撑柱。
* **步骤过渡**：
  * `1`：`.tub` 底座出现。
  * `2`：`.div1` 隔板插入底座中心。
  * `3`：`.div2` 绕 Z 轴旋转 90 度并插下，与 `.div1` 形成十字相交。
  * `4`：`.tree` 支撑从一旁拔地而起，托举起顶部。

#### Lesson 08: 旋转陀螺大赛
* **零件与面**：
  * `.disc`：直径 80px, 厚 12px 的 3D 深色金属质感陀螺圆盘。
  * `.pin`：直径 16px, 高 40px 的中心轴，且利用 CSS `box-shadow` 在边缘阵列形成红色的配重虚影。
* **步骤过渡**：
  * `1`：圆盘与中心轴同心显示。
  * `2`：两零件合并，轴心锁死。
  * `3`：边缘添加红色配重环（光圈浮现）。
  * `4`：陀螺整体翻转 180 度（`rotateX(-180deg)`），展示防下坠打印姿态。

#### Lesson 09: 镂空灯罩设计
* **零件与面**：
  * `.cyl`：直径 70px, 高 100px 的 3D 金黄色微透光镂空圆柱外壳。
  * `.hole`：贴合在灯罩上的菱形全息红光斑。
* **步骤过渡**：
  * `1`：暖黄色圆柱外壳出现。
  * `2`：红色切刀 `.hole` 贴合。
  * `3`：旋转中心标出。
  * `4`：阵列环绕复制出 8 个切刀，并在灯罩内壁形成等距的透光切口（镂空花纹亮起）。

#### Lesson 10: 关节可动小人
* **零件与面**：
  * `.box`：60x60x60px 半透明灰色 3D 球窝体。
  * `.ball`：直径 40px 3D 蓝色球形球头关节。
* **步骤过渡**：
  * `1`：方形身体块出现。
  * `2`：关节窝开孔（外层圆角及下凹）。
  * `3`：球头滑入中心。
  * `4`：场景绕 X 轴倾斜，通过半透明剖面与高亮红虚线虚框，展示球头与球窝的 0.3mm 配合间隙。

#### Lesson 11: 创意手机支架
